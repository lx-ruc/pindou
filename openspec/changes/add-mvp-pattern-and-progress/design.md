## 关键设计决策

### 决策 1：色号网格是通用枢纽
两个图纸入口（拍照生成 / 上传已有图纸）都产出同一个 rows×cols 色号网格；下游（推荐顺序、进度、未来的拍照识别）只消费网格、不关心来源。网格是一等公民数据结构，`Project` 持久化的就是网格 + 元数据（`source_type` 仅记录来源）。切换品牌时网格的 HEX 不变、只重映射色号——这是"实时重映射 < 500ms"可行的原因。

### 决策 2：像素化用主导色，不用均值
均值会在红黄边界算出橙色（图中不存在的中间色）。采用"降采样到 ~8×8 + 4bit/通道直方图众数"：众数是真实观察到的量化色，避免发灰；每格仅 64 个采样，廉价。bin 宽与采样数是可调旋钮，无理论最优，Phase 1 用真实图实测定（见 tasks 3.7）。不上 k-means/median-cut（留到 CIEDE2000 阶段）。

### 决策 3：推荐顺序 = 色批量 + 蛇形，非 TSP
单豆放置耗时（~2s）占绝对大头且与顺序无关；手部移动只占 ~5%。真正可省的是换色开销（~8s/次 × 12-30 色）。故按色批量（频率降序）最小化换色，色内蛇形（boustrophedon）避免回头。"最快"是 overclaim，文案统一用"推荐顺序"。

### 决策 4：进度追踪改用手动点选（MVP），拍照识别降级 P2
原计划 MVP 含拍照识别，但其 CV 重、需 spike、准确率受光照影响、撞小程序 2MB 包体限制。手动点选零 CV、100% 准确、与推荐路线天然合一（高亮下一颗 → 点击 → 前进）。故 MVP 进度 = 手动点选 + 引导；拍照识别作为之上的可选核对层，独立成下一变更。方案 Z spike 不再阻塞 MVP。

### 决策 5：每豆直接标注真实色号 + 采购清单图例
每颗豆上直接印当前品牌的真实色号（如 MARD `A01`、漫漫 `IC04`），字号按色号长度（1-4 位）自适应缩放，黑白按底色亮度切换。图例为采购清单（色号 → 色块 → HEX → 数量，按用量降序）。色号随品牌切换同步刷新（豆面需重绘，但仍只重映射、不重像素化）。导出按 29×29 板分页（每页网格 + 本页采购清单），单张 A4 标注全图不可读。

> 注：早期方案曾考虑"抽象符号 + 图例解码"（参照 pixel-beads），但用户明确要求每豆显示真实色号——色号本身即精确标识，无需额外符号层。已据此调整。

### 决策 6：上传已有图纸走同一管线，加"实际尺寸"输入
已是像素图的源图若按普通照片重像素化，源块边界与重画网格错位 → 串色。故上传路径加一行"原图实际尺寸"输入，按该尺寸精确切分、块内直接取代表色（不重采样）。自动块检测留后续。带精确色号的图纸会被 re-map 到 291 色（接受轻微偏色），不做图例 OCR。

### 决策 7：单格改色是品牌内覆盖层
编辑存 `{cell: overrideCode}`，不烙进网格——否则切品牌会冲掉用户编辑。切品牌清空覆盖并提示（MVP 简单方案；跨品牌连续的"颜色意图"覆盖留后续）。

### 决策 8（为 P2 预留）：方案 Z spike 五验证点
拍照识别进 P2 前需先跑 2-3 天去风险 spike：C1 `getImageData` 在小程序 Canvas 2D 可用；C2 全尺寸照片性能 < 5s（含预降采样）；C3 四点双线性 warp 几何精度；C4 逐格色差分正确/错误/空孔三类（含 EXIF、空孔检测）；C5 像素读取隔离平台层、核心纯 TS。短路逻辑：C1 失败 → 客户端优先破产需上云。详见 explore 讨论记录。

## 数据模型改动

- `Board` + `peg_inset`（钉网格边距，供拍照识别对齐补偿，P2 用）
- `Project` + `source_type`(generate/upload) + 多板状态 + `route_state`{current_segment, elapsed_per_segment, bead_rate} + `placed[][]` + 编辑覆盖层 `{(r,c): overrideCode}`
- `Palette` 子集数据（96/144/168）缺失 → MVP 仅全 291 色

## 平台顺序

图纸生成 + 进度追踪无 CV、无 spike 阻塞 → 直接 H5 快速迭代 → Taro 编译微信小程序。仅拍照识别（P2）需先在真机跑 spike。

## 视觉语言（由 Desktop 原型确立）

像素艺术（pixel-art）+ neobrutalism（粗黑描边、硬偏移投影、饱和原色）+ Fredoka/Nunito 圆润字体；钉板圆点纹理背景；豆子以扁平方块渲染（无圆豆选项，纯像素块）。配色：手工纸奶米色底 + 拼豆橙(#F77F00) 主色 + 钉板蓝/绿点缀。
