# 拼豆智能助手 · 品牌 Logo

「智能拼豆」四字 2×2 像素拼豆 logo，底色米白 `#FBF6EC`，字色用品牌四色（智橙·能红·拼蓝·豆绿）。

## 文件

| 文件 | 尺寸 | 用途 |
|------|------|------|
| `pindou-logo-4color.png` | 768×768 方形 | 平台上传（微信小程序会自行裁圆） |
| `pindou-logo-4color-512.png` | 512×512 方形 | 通用上传尺寸 |
| `pindou-logo-4color-squircle.png` | 768×768 squircle（圆角 22%，外角透明） | 展示 / 通用 |

## 规格参数

- 像素网格：48×48（每字 24×24）
- 字体：Hiragino Sans GB **W6**（粗体），采样阈值：一格内深色像素占比 > 30% 即判为"字"
- 工艺：canvas/PIL 渲染文字 → 采样像素化（与图纸生成器同算法）→ 按象限上四色
- 配色取自 `data/colorSystemMapping.json` 品牌体系：`#F77F00` `#E63946` `#2D6CDF` `#06A77D`

## 重新生成 / 调参

打开 `logo-generator.html`（浏览器）可实时预览三种字色（墨黑 / 四色 / 主橙）× 三档网格（32/48/64）并下载 PNG。

底色、四色、网格、阈值、字体粗细均可调；要改告诉我。
